### 🧠 AmdoxAI — Emotion Intelligence Platform
*Three Month Internship Project · Amdox Technologies · Group 3, Batch 4.2 · 2026*

[![Live Demo](https://img.shields.io/badge/🚀_Live_Demo-amdoxai--platform.vercel.app-00F5FF?style=for-the-badge)](https://amdoxai-platform.vercel.app)
[![API Docs](https://img.shields.io/badge/📖_API_Docs-Swagger_UI-4ade80?style=for-the-badge)](https://amdox-backend.onrender.com/docs)
[![GitHub](https://img.shields.io/badge/💻_Source-GitHub-181717?style=for-the-badge)](https://github.com/abhinashkumarin/amdoxai-platform)

</div>

> AI-powered HR Stress Monitoring SaaS — detects employee emotions via **Text (Hinglish NLP)**, **Voice (wav2vec2 transformer)**, and **Face (DeepFace + RetinaFace)** in real-time.

```
📦 Stack: Python · FastAPI · React 18 · TensorFlow · PyTorch · Supabase · Clerk · Chrome MV3
🎯 Problem: 76% Indian employees face burnout. HR has no early warning system.
💡 Solution: Tri-modal AI fusion engine + real-time WebSocket HR dashboard + Chrome extension
```

| Feature | Technology |
|---------|-----------|
| Text Emotion (Hinglish) | TextBlob NLP + Custom Keyword Engine |
| Voice Emotion | Facebook wav2vec2-base-superb-er |
| Face Emotion | DeepFace + RetinaFace |
| Real-time Alerts | FastAPI WebSocket + Redis Pub/Sub |
| Multi-tenant DB | Supabase PostgreSQL + Row Level Security |
| Auth | Clerk JWT (RBAC: employee/hr/admin) |
| Passive Monitoring | Chrome Extension (Manifest V3) |
